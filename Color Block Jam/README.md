# Color Block Jam

A fun and challenging puzzle adventure where your creativity and logic skills are key to unlocking new levels!

## üéÆ How to Play

1.  **Goal**: Clear the board by sending colored blocks to their matching doors.
2.  **Controls**: Click/Touch and drag a block to slide it.
3.  **Rules**:
    *   Blocks can only move in the direction of the drag (Horizontal or Vertical).
    *   Blocks cannot move through other blocks or walls.
    *   To clear a block, slide it towards a **matching colored door** on the edge of the grid.
    *   If the path is clear and the block touches the matching door, it will exit the board.

## üöÄ How to Run

### Locally
Simply open the `index.html` file in your web browser.

### Using a Local Server (Recommended)
If you have Python installed, you can run a simple HTTP server:

1.  Open your terminal.
2.  Navigate to the project directory:
    ```bash
    cd "Color Block Jam"
    ```
3.  Run the server:
    ```bash
    python3 -m http.server 8080
    ```
4.  Open your browser and go to `http://localhost:8080`.
